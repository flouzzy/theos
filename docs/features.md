# Features Documentation

Le Rocher Académie provides a comprehensive learning management system with the following features:

## For Students

### Course Enrollment & Learning
- Browse available courses
- Enroll in courses
- Track progress through modules and lessons
- Mark lessons as complete
- Comment on lessons for discussion
- Receive certificates upon completion

### Personal Dashboard
- View enrolled courses
- Track overall progress
- Access upcoming cohorts
- Manage personal calendar
- View recent notifications

### Note Taking
- Create personal notes during lessons
- Organize notes by course/lesson
- Search and filter notes
- Export notes (future feature)

### Gamification
- Earn badges for achievements
- Track learning streaks
- Unlock special content
- Compete on leaderboards (future)

### Calendar & Scheduling
- View cohort schedules
- Get reminders for live sessions
- Sync with personal calendar

## For Instructors

### Course Management
- Create and organize courses
- Structure content into modules and lessons
- Upload multimedia content
- Set prerequisites
- Publish/unpublish courses

### Content Creation
- Rich text editor for lessons
- Video embedding
- File attachments
- Code snippets
- Quizzes (future)

### Student Management
- View enrolled students
- Track student progress
- Review completion rates
- Moderate comments

### Analytics
- Course engagement metrics
- Completion rates
- Student feedback
