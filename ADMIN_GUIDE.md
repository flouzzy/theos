# Guide Administrateur Theos ⚙️

Ce guide vous aide à gérer votre LMS Theos pour offrir la meilleure expérience possible à vos apprenants.

## 🎨 Personnalisation
### Modifier le Logo
Dans votre interface admin, accédez à **Paramètres > Apparence**. Vous pouvez y téléverser votre propre logo qui sera affiché sur toutes les pages de l'application.

## 👥 Gestion des Apprenants & Promos
- **Promotions (Cohortes)** : Les sections "Calendrier" et "Évaluations" sont dynamiques. Elles ne s'affichent automatiquement que pour les utilisateurs rattachés à une promotion active.
- **Attribution** : Assurez-vous d'ajouter vos élèves dans les bonnes promotions pour qu'ils aient accès à tout leur contenu.

## 📚 Gestion du Contenu
- **Parcours** : Créez, modifiez ou publiez vos parcours dans la section "Cours".
- **Statut** : N'oubliez pas de passer vos leçons et cours au statut **"Published"** pour qu'ils soient visibles par les élèves.

## 🚀 Astuces Admin
- Vérifiez régulièrement les statistiques de progression pour identifier les élèves qui pourraient avoir besoin d'aide.
- Utilisez la commande `app:update-lesson-content` si vous avez besoin de rafraîchir le contenu de manière globale.
